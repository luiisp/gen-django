Docs Gen Template Django

O Gen Django é um projeto Django que fornece uma estrutura inicial completa com operações CRUD e Confirmação por email para servir como um ponto de partida para outros projetos. Este documento descreve a estrutura do projeto, suas funcionalidades e como utilizá-lo.

Features

  • CRUD de usuários
  • Interface Responsiva
  • Confirmação de conta por email
  • Estrutura de projeto modular e escalável
